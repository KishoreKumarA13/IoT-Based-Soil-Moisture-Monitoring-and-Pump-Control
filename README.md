\# IoT-Based Soil Moisture Monitoring \& Pump Control



\## Project Overview



This project presents an IoT-Based Soil Moisture Monitoring \& Pump Control system developed using the ESP32 microcontroller, Soil Moisture Sensor, Relay Module, and the Blynk IoT platform. The system continuously monitors soil moisture levels and automatically controls a water pump to maintain optimal soil conditions. It also enables remote pump control through the Blynk mobile application.



\---



\## Features



\- Real-time soil moisture monitoring

\- Automatic water pump control

\- Manual pump control using Blynk mobile app

\- ESP32 Wi-Fi connectivity

\- Relay-based pump switching

\- Smart irrigation solution

\## Working Principle



The ESP32 continuously reads the soil moisture sensor value.



When the soil moisture falls below the predefined threshold, the relay module switches ON the water pump automatically.



The user can also manually control the water pump through the Blynk mobile application.
